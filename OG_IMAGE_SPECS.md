# Open Graph Image Specifications

## Требования к OG изображению

Для правильного отображения в социальных сетях нужно создать изображение со следующими характеристиками:

### Размеры
- **Размер**: 1200×630 пикселей
- **Формат**: JPG или PNG
- **Соотношение сторон**: 1.91:1
- **Имя файла**: `og-image.jpg`
- **Расположение**: `/public/og-image.jpg`

### Рекомендации по дизайну

1. **Логотип Art Garage** - разместите в центре или слева
2. **Слоган/Текст**:
   - "Креативное рекламное агентство полного цикла"
   - "Creative full-cycle agency"
3. **Визуальные элементы**:
   - Можно использовать примеры работ в виде коллажа
   - Фирменные цвета Art Garage
   - Градиенты или фон
4. **Текст должен быть читаемым** - используйте контрастные цвета
5. **Избегайте мелких деталей** - изображение будет уменьшено

### Безопасные зоны
- Оставьте отступы по краям минимум 40px со всех сторон
- Центральная зона 1200×600px - самая важная (некоторые платформы могут обрезать края)

### Где будет использоваться
- Facebook
- Twitter/X
- LinkedIn
- Telegram
- WhatsApp
- Другие социальные сети при шеринге ссылок

### Как проверить
После создания изображения можно протестировать как оно будет выглядеть:
- Facebook Sharing Debugger: https://developers.facebook.com/tools/debug/
- Twitter Card Validator: https://cards-dev.twitter.com/validator
- LinkedIn Post Inspector: https://www.linkedin.com/post-inspector/

## Альтернатива: Автоматическая генерация

Если нужно создать OG изображение программно, можно использовать:
- `@vercel/og` - для автоматической генерации OG изображений в Next.js
- Создать API route, который генерирует изображения динамически

Пример: `/app/api/og/route.tsx`
