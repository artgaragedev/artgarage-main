# Как получить Service Role Key из Supabase

## Шаг 1: Откройте Supabase Dashboard
Перейдите на https://supabase.com/dashboard

## Шаг 2: Выберите ваш проект
Выберите проект: **rlcpynwvpgubxvsuvkew**

## Шаг 3: Откройте настройки API
1. В левом меню нажмите **Settings** (Настройки)
2. Выберите **API**

## Шаг 4: Найдите Service Role Key
1. Прокрутите вниз до раздела **Project API keys**
2. Найдите ключ с названием **service_role** (он должен быть скрыт)
3. Нажмите на иконку глаза, чтобы показать ключ
4. Скопируйте этот ключ

## Шаг 5: Добавьте в .env.local
Откройте файл `.env.local` и замените:
```env
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key-here
```

на

```env
SUPABASE_SERVICE_ROLE_KEY=скопированный-ключ
```

## Шаг 6: Перезапустите dev сервер
```bash
# Остановите текущий сервер (Ctrl+C)
# Запустите снова:
npm run dev
```

## ВАЖНО! ⚠️
- Service Role Key обходит все правила безопасности (RLS)
- НИКОГДА не используйте его на клиентской стороне
- НИКОГДА не коммитьте его в Git
- Используйте только в серверных API routes

После этого форма заказа заработает без проблем с RLS!
